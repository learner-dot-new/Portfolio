Algorithm

Match-Making Engine

The logic behind "Spaces" 1-on-1 discovery, prioritizing compatibility and safety.

Queue Management

A global spacesQueue stores candidates. The engine iterates through the queue using a Weighted Matching Score.

  • Interest Intersection: +2 pts per match
  • Language Fluency: +1 pt match
  • Gender Mismatch: Immediate disqualification

Governance

Moderation & Safety

A professional suite of tools for maintaining community standards and safety.

Real-time Suspension

Admins can force-terminate any room or block users instantly. Blocking triggers a socket.disconnect() and persistent database flag.

Reporting Pipeline

Users can file reports against rooms or profiles. Reports are aggregated in the Admin Hub with context snapshots (room type, host, time).

Admin Warnings

Send direct, non-ignorable warnings to users in active rooms. Warnings are delivered via a high-priority socket event admin-warning.
